Netflix Data Analytics Dashboard – Tableau Project
Project Overview

This project presents an interactive Netflix Analytics Dashboard developed using Tableau to analyze and visualize content trends within Netflix’s dataset. The goal was to transform raw data into structured, meaningful insights that support data-driven understanding of streaming content distribution and patterns.

The dashboard provides a comprehensive view of Netflix’s content library across multiple analytical dimensions.

Problem Statement

Streaming platforms generate vast amounts of content data. However, extracting meaningful insights from this data requires structured visualization and analytical interpretation.

This project aims to answer key business-oriented questions such as:

What is the distribution between Movies and TV Shows?

How is Netflix content distributed across countries?

Which genres dominate the platform?

How has content growth evolved over the years?

What rating classifications are most common?

Key Insights Generated

Comparative analysis of Movies vs TV Shows

Country-wise content distribution

Ratings classification trends

Top 10 most represented genres

Year-wise content addition trends

Dashboard Features

Interactive filters for dynamic exploration

Clean and structured layout for clarity

Visual storytelling approach for better interpretation

KPI-focused design to highlight critical insights
